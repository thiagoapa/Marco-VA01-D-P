# Controle de Estoque de Produtos (Java)

Este projeto é uma aplicação simples em Java para gestão de stock, desenvolvida como parte da atividade Marco-VA01. A aplicação permite o registo de um produto e a atualização da quantidade em stock após uma venda, com validação de disponibilidade.

## 🚀 Funcionalidades

- Registo de Nome, Preço e Quantidade Inicial.
- Processamento de venda com redução automática de stock.
- Validação: Impede vendas se a quantidade solicitada for superior ao stock disponível.
- Exibição de dados formatados (R$ e unidades).

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** Java 21 (ou superior)
- **Paradigma:** Programação Orientada a Objetos (POO)
- **IDE:** IntelliJ IDEA

## 📂 Estrutura do Projeto

- `src/entities/Produtos.java`: Classe que contém os atributos do produto e a lógica de negócio (venda).
- `src/application/Program.java`: Classe principal que gere a interação com o utilizador via consola.
